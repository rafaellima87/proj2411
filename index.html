HTML

O HTML (Linguagem de Marcação de HiperTexto) é a espinha dorsal de qualquer site e o bloco de construção mais básico da web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que permite estruturar e organizar o conteúdo de uma página web, como textos, imagens e links.

CSS

O CSS (Cascading Style Sheets) é uma linguagem fundamental para o desenvolvimento web, utilizada para estilizar e controlar a apresentação visual de documentos escritos em HTML [1]. Ele permite separar a estrutura do conteúdo (HTML) da sua aparência, como cores, fontes, tamanhos, espaçamento e layout [1].

Para adicionar uma borda a uma imagem com CSS, use a propriedade border e defina a largura, estilo e cor da borda. Por exemplo, border: 2px solid black; adicionará uma borda preta sólida de 2 pixels ao redor da imagem.
